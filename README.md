# 賽博龐克貓咪塔防 (Cyberpunk Cat Defense)

一款融合賽博龐克風格與可愛貓咪元素的多平台塔防遊戲，在霓虹閃爍的未來都市中，守護你的賽博貓咪基地！

## 🎮 遊戲特色

- **賽博龐克視覺風格**：霓虹光效、數位故障效果、科技感十足的 UI
- **可愛貓咪基地**：會根據血量變化表情的賽博貓咪基地
- **多樣化敵人**：四種不同類型的敵機，包含 BOSS 戰
- **升級系統**：豐富的武器升級選項，打造專屬戰鬥風格
- **特效系統**：愛心爆炸效果、六邊形粒子、華麗的能量大招
- **連擊系統**：維持連殺獲得額外分數
- **能量系統**：累積傷害釋放強大的範圍攻擊

## 🎯 支援平台

- **🖥️ 桌面 Web 版**：完整的遊戲體驗，支援滑鼠操作
- **📱 移動 Web 版**：觸控優化，適配手機和平板
- **📲 Android App**：原生應用體驗，優化性能
- **🔧 遊戲編輯器**：視覺化配置編輯工具

## 🕹️ 操作方式

- **滑鼠/觸控**：點擊螢幕任意位置進行攻擊
- **自動瞄準**：基地會自動攻擊最近的敵人
- **能量大招**：能量條滿時自動觸發華麗的全屏攻擊

## 🚀 遊戲玩法

1. 敵機會從四面八方進攻你的賽博貓咪基地
2. 點擊螢幕協助基地攻擊敵人
3. 擊敗敵人獲得金幣和經驗
4. 每波結束後選擇升級項目強化基地
5. 挑戰更高的波數，保護賽博貓咪！

## 🛠️ 技術架構

### 核心技術
- **前端技術**：HTML5 Canvas + 原生 JavaScript ES6+

- **性能優化**：物件池、LOD 渲染、動態品質調整
- **響應式設計**：適配桌面、平板、手機各種尺寸

### 架構設計
- **模組化架構**：基於 `src/` 目錄的現代化結構
- **平台適配**：統一核心邏輯，平台特定優化

## 🚀 快速開始

### 1. 克隆專案
```bash
git clone https://github.com/Chuanyin1202/cyberpunk-cat-defense.git
cd cyberpunk-cat-defense
```

### 2. Web 版本開發
```bash
# 方法一：使用專案提供的腳本
./server-utils/start_server.sh

# 方法二：使用 Python 開發伺服器
python3 -m http.server 8080
```

**訪問地址：**
- 🖥️ 桌面版：`http://localhost:8080/`
- 📱 移動版：`http://localhost:8080/mobile/`
- 🔧 編輯器：`http://localhost:8080/editor/`

### 3. Android App 開發
```bash
# 安裝依賴
npm install

# 同步到 Android 專案
npm run sync

# 編譯 APK
npm run android:release
```
